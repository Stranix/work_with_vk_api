# Публикация комиксов в группе VK
Получение случайного комикса с сайта [xkcd.com](https://xkcd.com/) с последующей публикацией в сообществе [Вконтакте](https://vk.com/).

## Как установить и использовать
1. Задать переменные окружения `VK_GROUP_ID`, `VK_ACCESS_TOKEN`.  
 `VK_GROUP_ID` - id группы в контакте (узнать можно [тут](https://regvk.com/id/)) куда будем постить комиксы.  
 `VK_ACCESS_TOKEN` - токен полученный по инструкции [тут](https://vk.com/dev/implicit_flow_user) 
2. Установить зависимости:
```
pip install -r requirements.txt
```
3. Запустить скрипт:
```
python3 main.py
```
При [запросе токена](https://vk.com/dev/implicit_flow_user) следует убрать параметр `redirect_uri`, 
необходимый пул прав (scope) указать как: `scope=photos,groups,wall`  

Результатом будет опубликованная запись с случайным комиксом в заданной группе.  

## Цель проекта
Код написан в образовательных целях на онлайн-курсе для веб-разработчиков [dvmn.org](https://dvmn.org/).
